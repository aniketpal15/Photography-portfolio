# PHO·TOS | Professional Photographer Portfolio

A premium, interactive photography portfolio website designed to showcase landscape, portrait, and architectural works with cinematic animations.

**Live Demo:** [https://advphotography.vercel.app/](https://advphotography.vercel.app/)
**GitHub Repository:** [https://github.com/aniketpal15/Photography-portfolio](https://github.com/aniketpal15/Photography-portfolio)

## ✨ Features
*   **3D Camera Preloader:** An immersive, custom-built camera lens animation that plays while the site initializes.
*   **Smooth Scrolling:** Integrated Lenis smooth scroll for a fluid, premium browsing experience.
*   **Advanced Animations:** Utilizes GSAP and ScrollTrigger for scroll-linked animations, parallax effects, and element reveals.
*   **Interactive Gallery:** Features an infinite scrolling marquee gallery with category filtering (Landscape, Portrait, Architecture, Street, and Wedding).
*   **Modern UI/UX:** Includes a custom cursor, mouse-follower, and 3D parallax hover effects on hero visuals.
*   **Glassmorphism Design:** Beautiful frosted-glass aesthetic applied to the Services and Testimonials sections.
*   **Working Contact Form:** Backend integration using Express and Nodemailer to handle client inquiries directly to your email.

## 🛠️ Tech Stack
*   **Frontend:** HTML5, CSS3, Vanilla JavaScript
*   **Build Tool:** Vite
*   **Animation & Scrolling:** GSAP, ScrollTrigger, Lenis
*   **Backend:** Node.js, Express.js, Nodemailer, CORS, dotenv
